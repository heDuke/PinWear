# PinWear

PinWear is a third-party Wear OS client based on the official Pinterest API, dedicated to providing a native Pinterest browsing experience for smartwatch users.

PinWear 是基于 Pinterest 官方 API 开发的第三方 Wear OS 客户端，致力于为智能手表用户提供原生的 Pinterest 内容浏览体验。
